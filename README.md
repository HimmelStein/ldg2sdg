# sentence to parts
A Python module to transform raw linguistic graph into spatial dependency graph

# Quick start
```
$ git clone https://github.com/Himmelstein/ldg2sdg
$ cd ldg2sdg
ldg2sdg $ make init
```

# Run test
```
ldg2sdg $ make test
```

# Generate documentation
```
ldg2sdg $ ./make_docu
```
Documentation is located at docs/html/
